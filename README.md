# VendorLink 🛒

VendorLink is a full-stack MERN marketplace platform.

The project is currently in active development.

---

## 🚀 Tech Stack

### Frontend
- React.js (Vite)
- Tailwind CSS
- Zustand (state management)
- Axios

### Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
